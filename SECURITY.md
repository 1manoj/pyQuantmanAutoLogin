# Security Policy

## Vulnerabilities

If you discover a security vulnerability within this project, please open a private security advisory or contact the maintainers. All security vulnerabilities will be promptly addressed.

## Reporting Security Issues

To report a security issue, please include the following information in your email:
- A description of the vulnerability
- Steps to reproduce the vulnerability (if applicable)
- Any relevant log files or screenshots that can assist in understanding the issue

## Security Guidelines

- Avoid sharing sensitive information publicly.
- Keep your dependencies up to date.
- Regularly review the project for security vulnerabilities.

Thank you for helping to make this project more secure!
